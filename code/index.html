Current projects

Past projects

  • optax: gradient processing and optimization library for JAX. I was a contributor.
  • JAXopt: hardware accelerated, batchable and differentiable optimizers in JAX. I was the lead developer.
  • lightning: a library for large-scale linear classification, regression and ranking in Python. I was the lead developer.
  • svmlight-loader: fast and memory-efficient svmlight / libsvm file loader. I was the co-lead developer.
  • scikit-learn: easy-to-use and efficient machine learning library for the Python scientific ecosystem. I was core developer.
  • tegaki: Chinese and Japanese handwriting recognition. I was the lead developer.
  • fantasdic: versatile dictionary application. I was the lead developer.
  • WikipediaFS: view and edit Wikipedia articles as if they were real files. I was the lead developer
